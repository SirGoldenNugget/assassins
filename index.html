Welcome to Assassin Party Game

Assassins is a game that can be played almost anywhere, but best done in semi-crowded environments. The goal of the game is to be the last remaining person alive and to assassinate whoever you are assigned.

Start Playing Now

Instructions & How to Play

  • Login/Register

    To start playing you must create an account and login. Try to make your password something easily rememberable for you might have to log back in more than once.

  • Join/Create A Lobby

    Once logged in, you can either join or create a lobby. To join a lobby, ask the party leader for the lobby code and enter it; creating a lobby will generate a code.

  • Start Playing

    That's it! Just click the start game button once everyone has joined the game and may the last individual standing be the winner. Make sure to confirm you kills.
